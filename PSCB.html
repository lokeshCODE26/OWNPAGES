The Trials of Numeria

The land of Numeria is home to five ancient challenges that test a programmer's skills in logic, computation, and problem-solving. To prove your worth, you must solve the following five trials:

The Enchanted Swap

The great magician of Numeria possesses two magical orbs, each holding a unique number. To unlock a hidden treasure, he must swap the values of these two orbs using a single spell.

Input Format:
Two space-separated integers A and B (-10⁶ ≤ A, B ≤ 10⁶).

Output Format:
Print the two numbers after swapping, separated by a space.

Example:
Input: 7 3
Output: 3 7

The Tower of Factoria

The kingdom of Factoria calculates the strength of its magical towers using the factorial of a given number n. Your task is to compute the factorial of n.

Input Format:
A single non-negative integer n (0 ≤ n ≤ 20).

Output Format:
Print the factorial of n.

Example:
Input: 5
Output: 120

The Mirror of Truth

In Palindoria, a string is considered magical if it reads the same forward and backward, ignoring spaces, punctuation, and capitalization. Your task is to determine whether a given string is a palindrome.

Input Format:
A string S (1 ≤ |S| ≤ 10⁶) containing letters, digits, spaces, and punctuation.

Output Format:
Print "palindrome" if the string is a palindrome. Otherwise, print "not a palindrome".

Example:
Input: Madam
Output: palindrome

The Leap Year Prophecy

The royal calendar of Numeria follows a special rule for leap years:
- A year is a leap year if it is divisible by 4.
- However, if it is divisible by 100, it is not a leap year unless it is also divisible by 400.

Input Format:
A single integer Y (1 ≤ Y ≤ 10⁶).

Output Format:
Print "leap" if the year is a leap year. Otherwise, print "not a leap year".

Example:
Input: 2000
Output: leap

The Greatest Numbers Duel

Two legendary numbers have challenged each other in a duel. Your task is to determine their Greatest Common Divisor (GCD) and Least Common Multiple (LCM).

Input Format:
Two space-separated integers X and Y (1 ≤ X, Y ≤ 10⁶).

Output Format:
Print two space-separated integers: GCD(X, Y) LCM(X, Y).

Example:
Input: 8 12
Output: 4 24
